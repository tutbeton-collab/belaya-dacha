# 🌿 Белая Дача — Сайт семейной фермы

Современный веб-сайт для семейной фермы натуральных овощей "Белая Дача", созданный на Next.js 14+ с использованием Tailwind CSS.

## 🚀 Быстрый старт

### Требования

- Node.js 18.17 или новее
- npm или yarn

### Установка

1. **Установите зависимости:**

```bash
cd belaya-dacha
npm install
```

2. **Запустите сервер разработки:**

```bash
npm run dev
```

3. **Откройте браузер:**

Перейдите по адресу [http://localhost:3000](http://localhost:3000)

## 📱 Секции сайта

1. **Hero** — Приветственная секция с заголовком и кнопками призыва к действию
2. **О ферме** — Информация о семейной ферме и методах выращивания
3. **Овощи** — Карточки с описанием выращиваемых овощей
4. **Преимущества** — Почему выбирают нас
5. **Оптовикам** — Информация для оптовых партнёров
6. **Галерея** — Фотографии с фермы
7. **Контакты** — Кнопки связи через MAX и Telegram
8. **Подвал** — Навигация и информация о ферме

## 📸 Добавление изображений

Для добавления реальных фотографий в галерею:

1. Поместите изображения в папку `public/images/`
2. Назовите их `gallery-1.jpg`, `gallery-2.jpg` и т.д.
3. Обновите компонент `Gallery.tsx` при необходимости
