# Белая Дача — Розыгрыш

Приложение для проведения розыгрышей среди участников.

## Функции

- ➕ Добавление участников
- 🗑 Удаление участников
- 🎲 Розыгрыш победителя
- 📜 История розыгрышей
- 💾 Локальное хранение (Room DB)

## Технологии

- Kotlin + Jetpack Compose
- Room Database
- MVVM Architecture
- Material3 + Navigation

## Сборка

```bash
./gradlew assembleDebug
```
